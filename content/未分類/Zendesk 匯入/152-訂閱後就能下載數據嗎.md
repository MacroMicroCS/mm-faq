感謝您在訂閱前確認，這樣我們才能確保您選到最合適的方案！\
MM Prime / MM Max 個人方案目前不包含原始數據（CSV）下載功能。\
但這兩個方案可以讓您：\
✓ 在 Toolbox 中自訂圖表、進行指標運算與視覺化\
✓ 將圖表匯出為圖片\
✓ 生成可分享的圖表連結\
\
如果您的工作需要原始數據下載或 API 串接，這部分屬於 [MM 企業方案](https://www.macromicro.me/subscribe#enterprise)，歡迎先參考專屬方案介紹內容，如有購買需求歡迎來信聯繫 business@macromicro.me。

Thank you for checking before subscribing — that helps us make sure you choose the right plan!\
\
MM Prime and MM Max individual plans do not currently include raw data (CSV) download functionality.\
\
However, these plans do allow you to:\
✓ Build custom charts in the Toolbox with indicator calculations and visualizations\
✓ Export charts as images\
✓ Generate shareable chart links\
\
If your work requires raw data downloads or API access, that falls under the [MM Enterprise Plan](https://en.macromicro.me/subscribe#enterprise). Feel free to review the plan details, and reach out to business@macromicro.me if you're interested in purchasing.
